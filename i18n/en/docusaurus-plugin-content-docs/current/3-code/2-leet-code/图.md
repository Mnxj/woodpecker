### 133. Clone Graph

```html
Input: adjList = [[2,4],[1,3],[2,4],[1,3]]
Output: [[2,4],[1,3],[2,4],[1,3]]
Explanation:
The graph has 4 nodes.
Node 1 has value 1 and neighbors: Node 2 and Node 4.
Node 2 has value 2 and neighbors: Node 1 and Node 3.
Node 3 has value 3 and neighbors: Node 2 and Node 4.
Node 4 has value 4 and neighbors: Node 1 and Node 3.
```

**Solution 1**

DFS — recursively create each node and its neighbors.
Start at node 1.
Empty → return.
Return the newly created node.
```js
var cloneGraph = function(node) {
    if(!node) return;
    const nodeMap = new Map();
    const dfs = (n) => {
        const newNode = new Node(n);
        newMap.set(n, newNode)
        (n.neighbors || []).forEach(neighbor => {
            if(!nodeMap.has(neighbor)){
                dfs(neighbor);
            }
            newNode.neighbors.push(newMap.get(neighbor));
        })
    }
    dfs(node);
    return nodeMap.get(node);
};
```
