### 77. Combinations

```json
Given two integers n and k, return all possible combinations of k numbers chosen from the range [1, n].
You may return the answer in any order.

Example 1:

Input: n = 4, k = 2
Output:
[
  [2,4],
  [3,4],
  [2,3],
  [1,2],
  [1,3],
  [1,4],
]
Example 2:

Input: n = 1, k = 1
Output: [[1]]

Constraints:
1 <= n <= 20
1 <= k <= n
```

**Solution 1**

1. `res` stores results; `queue` stores matching values — push to `res` when length equals k
2. Use recursion `helper(index, queue)` —
the `helper` method acts as the execution context; a `for` loop pushes into the queue in order
```js
/**
 * @param {number} n
 * @param {number} k
 * @return {number[][]}
 */
var combine = function(n, k) {
        const res = [];
        const dfs = (start, arr) => {
            for(let i = start; i <= n; i++){
                const newarr = arr.concat(i);
                if(newarr.length === k) {
                    res.push(newarr);
                } else {
                    dfs(i + 1, newarr);
                }
            }
        }
        dfs(1, []);
        return res;
    };
```
